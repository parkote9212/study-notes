# Cursor 단축키 (Mac 기준) — 키보드만으로 코딩 & 실행

  

맥에서 키보드만 사용해 코드 쓰고 실행하는 데 필요한 단축키만 정리했습니다.

  

---

  

## 1. 실행 & 터미널

  

| 단축키 | 동작 |

| --------------------------- | ------------------------------------------- |

| **⌃ Control + ⇧ Shift + J** | 터미널 열기 / 닫기 (한글 키보드 권장) |

| **⌃ Control + `** | 터미널 열기 / 닫기 (영문 키보드) |

| **⌃ Control + ⇧ Shift + `** | 새 터미널 탭 |

| **⌘ Cmd + ⇧ Shift + P** | 명령 팔레트 (여기서 "Run Python" 검색 가능) |

| **⌘ Cmd + ⇧ Shift + B** | 빌드/실행 (설정에 따라 현재 파일 실행) |

  

**Python 실행 (키보드만):**

  

1. **⌘ Cmd + ⇧ Shift + P** → `Run Python File` 입력 → Enter

또는

2. **⌃ Control + ⇧ Shift + J** 로 터미널 연 뒤

`python chap01/basics.py` 입력 → Enter

  

---

  

## 2. 파일 & 탐색

  

| 단축키 | 동작 |

| ----------------------- | -------------------------------------- |

| **⌘ Cmd + P** | 파일 이름으로 열기 (빠른 이동) |

| **⌘ Cmd + ⇧ Shift + E** | 탐색기(사이드바) 포커스 |

| **⌘ Cmd + B** | 사이드바 보이기/숨기기 |

| **⌘ Cmd + 1, 2, 3...** | 왼쪽 1번, 2번, 3번… 패널로 포커스 이동 |

| **⌘ Cmd + J** | 하단 패널(터미널 등) 열기/닫기 |

  

---

  

## 3. 편집 (코드 작성)

  

| 단축키 | 동작 |

| ------------------------------ | ------------------------------------------------------ |

| **⌘ Cmd + S** | 저장 |

| **⌘ Cmd + Z** | 실행 취소 |

| **⌘ Cmd + ⇧ Shift + Z** | 다시 실행 |

| **⌘ Cmd + D** | 같은 단어 다음 occurrence 선택 (연속 누르면 다중 선택) |

| **⌘ Cmd + /** | 현재 줄(들) 주석 토글 |

| **⌥ Option + ↑ / ↓** | 현재 줄 위/아래로 이동 |

| **⌥ Option + ⇧ Shift + ↑ / ↓** | 현재 줄 위/아래로 복사 |

| **⌘ Cmd + X** | 줄에 내용 없으면 줄 잘라내기 (삭제) |

| **⌘ Cmd + Enter** | 아래에 새 줄 추가 후 그 줄로 이동 |

| **⌘ Cmd + ⇧ Shift + K** | 현재 줄 삭제 |

  

---

  

## 4. 이동 & 선택

  

| 단축키 | 동작 |

| ----------------------- | -------------------------------------------------------- |

| **⌘ Cmd + ← / →** | 줄 맨 앞 / 맨 뒤 |

| **⌥ Option + ← / →** | 단어 단위 왼쪽 / 오른쪽 |

| **⌘ Cmd + ↑ / ↓** | 파일 맨 위 / 맨 아래 |

| **⌘ Cmd + G** | 특정 줄로 이동 (줄 번호 입력) |

| **⌘ Cmd + L** | 현재 줄 선택 |

| **⌘ Cmd + ⇧ Shift + L** | 같은 문자열 모두 선택 후 다중 커서 |

| **⌥ Option + 클릭** | 여러 위치에 커서 (클릭 대신 **⌘ Cmd + D** 연타로도 가능) |

  

---

  

## 5. 찾기 & 바꾸기

  

| 단축키 | 동작 |

| ----------------------- | ------------------------------------ |

| **⌘ Cmd + F** | 현재 파일에서 찾기 |

| **⌘ Cmd + H** | 현재 파일에서 찾기 + 바꾸기 |

| **⌘ Cmd + ⇧ Shift + F** | 전체(워크스페이스)에서 찾기 |

| **⌘ Cmd + G** | 찾기에서 다음 결과 (F 열린 상태에서) |

| **⌘ Cmd + ⇧ Shift + G** | 찾기에서 이전 결과 |

  

---

  

## 6. 탭 & 에디터

  

| 단축키 | 동작 |

| ------------------------- | ---------------------- |

| **⌘ Cmd + N** | 새 파일 |

| **⌘ Cmd + W** | 현재 탭 닫기 |

| **⌘ Cmd + ⇧ Shift + T** | 방금 닫은 탭 다시 열기 |

| **⌘ Cmd + Tab** | 다음 탭 |

| **⌘ Cmd + ⇧ Shift + Tab** | 이전 탭 |

| **⌘ Cmd + \\** | 에디터 좌우 분할 |

  

---

  

## 6-1. 새 파이썬 파일 만들기 (코드 치다가 바로 새 .py 만들 때)

  

| 순서 | 단축키 | 동작 |

| ---- | --------------------------- | ------------------------------------------------------------------ |

| 1 | **⌘ Cmd + N** | 새 파일(Untitled) 열기 |

| 2 | **⌘ Cmd + K** 누른 뒤 **M** | 언어 모드 → `Python` 선택 (또는 우하단 "Plain Text" 클릭 → Python) |

| 3 | **⌘ Cmd + S** | 저장 → `chap01/이름.py` 처럼 경로+이름 입력 |

  

**한 줄 요약:** **⌘ Cmd + N** → **(⌘ Cmd + K, M)** → Python 선택 → **⌘ Cmd + S** → `이름.py` 저장

  

---

  

## 7. 자주 쓰는 조합 (코드 쓰고 실행까지)

  

1. **파일 열기** → **⌘ Cmd + P** → `basics` 입력 → Enter

2. **코드 수정** → **⌘ Cmd + S** 저장

3. **터미널 열기** → **⌃ Control + `**

4. **실행** → `python chap01/basics.py` 입력 → Enter

5. **에디터로 돌아오기** → **⌘ Cmd + 1** (또는 **⌃ Control + `** 한 번 더로 터미널 닫기)

  

**Run Python File 한 번에:**

**⌘ Cmd + ⇧ Shift + P** → `Run Python File` → Enter

  

---

  

## 8. 명령 팔레트 (⌘ Cmd + ⇧ Shift + P)

  

여기서 검색해서 실행할 수 있는 것들:

  

- `Run Python File in Terminal` — 현재 파이썬 파일 실행

- `Save` — 저장

- `Toggle Terminal` — 터미널 열기/닫기

- `Go to Line` — 줄 번호로 이동

- `Format Document` — 문서 포맷 (들여쓰기 등)

  

Mac 기준만 넣었고, 터미널·실행·편집 위주로만 정리했습니다. 더 필요하면 Cursor 메뉴 **Help → Keyboard Shortcuts Reference**에서 PDF로 전체 목록 볼 수 있습니다.
